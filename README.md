# Task 2: Exploratory Data Analysis (EDA) & Business Intelligence

## Objective
Perform Exploratory Data Analysis (EDA), answer business questions using SQL, and create an Excel dashboard.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQLite
- Microsoft Excel

## Key Insights
- Laptop generated the highest revenue.
- Electronics is the top-performing category.
- Patna generated the highest city-wise revenue.
- March recorded the highest monthly sales.
- Quantity and Unit Price strongly influence Total Sales.

## Deliverables
- EDA Notebook
- SQL Analysis
- Excel Dashboard
